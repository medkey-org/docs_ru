# Установка на CentOS

## Установка на CentOS 7

Инсталляция медицинской информационной системы Медкей производится на ОС CentOS 7 без графического интерфейса. Установка предполагает инсталляцию PHP 7.2 из внешнего репозитория пакетов \(remi-php72\).

### Исходные условия

* Установлена «чистая» ОС CentOS 7;
* Имеется root-доступ \(sudo-доступ\); вызовы `sudo` в листинге опущены;
* Подключён репозиторий пакетов в `/etc/yum.conf` либо `/etc/yum.repos.d/*`\(с диска, официального репозитория пакетов CentOS либо его зеркала\).

### Шаги по установке

### **1.Подключение репозитория пакетов PHP 7.2**

1. Установить утилиты: `yum install yum-utils`
2. Подключить репозиторий: `yum-config-manager --enable remi-php72`

### **2. Установка веб-сервера Apache Httpd 2.4**

1. Установить Apache Httpd 2.4: `yum install httpd`

### **3. Установка СУБД**

1. Установить PostgreSQL сервер и клиент: `yum install postgresql-server postgresql-client postgresql-contrib`
2. Инициализировать кластер БД: `postgresql-setup initdb`
